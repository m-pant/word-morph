Мне нужно добавить поддержку словосочетаний, необходимо сохранить текущую логику по работе с одтельными словами (описанную в параметрах) при этом должно создаваться слововочетанаия. Параметры
 - число слов в словосочетании
 - количество букв пропущенное в каждом слове
 - поддрежка режима пропуска букв во всем словосочетании ( то есть в одном слове могут быть пропущенны несколько букв а в дргом ни одной
 - словосочетания должны быть осознанными а не случайными ( пример бурый медведь, медленный осел, красивая ягодная поляна и т д) Не допускается с нарушением логиги ( утюжный индюк, раскрпощенный трава и т.д) По этому пункуту нужно составить план реализации и лучшие практики
 - поддрежка возраста - в зависимости от заданного возраста должны выдаваться релевантные слосочетания, например для ребенка 7 лет не должны выдаваться такие как экзистенциональная сингулярность
